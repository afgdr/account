Настройка Google OAuth

Для работы приложения необходимо:

  1. Создать OAuth client в Google Cloud Console
  2. Добавить в Authorized JavaScript origins: https://delta-exploits.pages.dev
  3. Заменить YOUR_CLIENT_ID в коде на ваш Client ID
